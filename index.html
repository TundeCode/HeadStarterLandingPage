Projects I've Work On

I have developed a variety of projects utilizing Python (with Tkinter), C++, and Thunkable (React Native framework).

Airline Project

Created a Program in Python that reads information about flights from Providence,RI to Orlando and it ask the user different question about their flight.

Hang Man

Developed a Python program that recreates the game of Hangman. Implemented functionality for the program to randomly select words from a predefined list, with the game being played in the terminal.

Contact List

Developed a contact list program in C++ utilizing a Trie data structure. Implemented methods for deletion, insertion, and search operations. The program allows users to create new contacts or search for existing ones within the list.

Profile App

Developed a full-stack mobile application using Thunkable. Configured Google Firebase as the backend, incorporating authentication and database functionality. The app enables users to create and verify accounts, upload profile pictures, and input personal details such as name, date of birth, favorite color, and favorite sport.

Study Sync

Co-developed an application called Study Sync. Collaborated with a team to develop a full-stack application using Thunkable. Designed a platform for users to connect and study together. Implemented the frontend using Thunkable, a React Native framework for mobile application development, and created responsive web interfaces utilizing HTML, JavaScript, and React.

Valid Sudoku Checker

Created a C++ program designed to validate a Sudoku Soultion by checking for the presence of duplicating numbers within rows,columns and subgrids of a 9 by 9 board.
